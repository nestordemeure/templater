# Business Card Cutter Templater

Web app to create [business card cutter](https://youtu.be/wGMH5a1iHFk) ready PDFs from card images.

## Usage

Open `index.html` in a browser or visit the [live app](https://nestordemeure.github.io/templater/).

## Features

- Upload front/back card images (drag-and-drop or file picker)
- Card presets (US/EU business cards, poker/bridge playing cards)
- Auto-centered layout with configurable grid and paper sizes
- Interactive preview: drag to reorder, click Ã— to delete
- Single back image automatically paired with all front images
- Fully client-side (images never leave your browser)
